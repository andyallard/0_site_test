Affiche publicitaire

Vous allez créer une affiche publicitaire pour « vendre » votre jeu. Cette affiche :

  • sera la taille d'une feuille de papier standard (8½ x 11).
  • comprendra un code-barres 2D (ex. code QR) avec un lien vers votre site web.
  • sera imprimée en couleur.
  • sera affichée dans l'école.

Chercher sur le web pour des exemples de comment créer une affiche publicitaire. Il n'est pas nécessaire de tout faire de zéro. Il y a des bon outils comme (ex. Canva) qui ne nécessite pas la création d'un compte.

Il y a plusieurs sites web qui permet de créer un code-barres 2D, comme cet outil crée par Alois Zingl.

Voir exemple ci-contre. Ce code-barres 2D vous amène vers la deuxième section de cette page (section sur la vidéo).
